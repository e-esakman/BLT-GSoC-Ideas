### BLT Flutter App Modernization & Mobile Contributor Companion (350 hours)

Repository: OWASP/BLT-Flutter

**Goal**

Deliver a modern, stable mobile application that makes bug reporting, contributor engagement, and BACON/badge tracking fast, accessible, and intuitive.

**Summary**

The BLT Flutter app will be rebuilt and refined into a polished mobile companion for contributors. It will allow users to report bugs instantly (with screenshots and offline drafts), view their BACON points, badges, and profile, and receive updates on activity. The focus is usability and accessibility — giving contributors a real-time, on-the-go BLT experience.

**Deliverables**
1. Updated Mobile UI/UX

- Fully refreshed interface (home, reporting, profile, leaderboard)
- Dark/light mode polish
- Smooth navigation and consistent theming

2. Fast Bug Reporting

- Screenshot capture + annotation
- Environment metadata extraction
- Autofill URL detection
- Offline mode (draft queue) → auto-sync when online

3. Contributor Dashboard

- BACON points display
- Badge showcase
- Leaderboard viewing
- Profile details

4. Notifications & Activity Feed

- Replies to reports
- BACON awarded
- New badges earned
- Issue/PR updates

5. Security & UX Enhancements

- Biometric authentication
- Local encryption of drafts and login tokens

6. Documentation & Release

- API usage documentation
- Developer setup guide
- Play Store + App Store release checklist
